# Análise de Dados de Vendas

Projeto desenvolvido para analisar dados de vendas e identificar padrões de faturamento e sazonalidade.

## Tecnologias
- Python
- Pandas
- Matplotlib
- Seaborn

## Principais Insights
- Crescimento das vendas no último trimestre do ano
- Produto Notebook com maior faturamento
- Evidência de sazonalidade no fim do ano
